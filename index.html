На сайте реализованы следующие функции:

- просмотр карточки товара,

- фильтрация товаров по размеру, цвету и цене на странице каталога,

- добавление товаров в корзину,

- просмотр корзины покупок,

- на странице администрирования есть возможность добавить и удалить товар, отредактировать информацию о товаре.

Для разработки сайта интернет-магазина мною были выбраны следующие технологии:

- для работы с макетом – онлайн-приложение Figma

- для верстки страниц сайта – HTML5, CSS3, SCSS

- для реализации функционала клиентской части – язык JavaScript

- для работы с базой данных и реализации серверной части, в том числе, приложения для администратора, - язык PHP

- для хранения данных - СУБД MySQL

- для стилизации администраторской части сайта используется Bootstrap 5.

- для соединения с базой данных применяются оба варианта: и PDO для работы с каталогом товаров, и MySQLi для регистрации и авторизации пользователя в системе.
